JavaScript란 무엇인가?


코딩 배우기 세 번째, 이 페이지는 JavaScript를 위한 페이지이며 연습용입니다.

자바 스크립트는 사용자와 웹 페이지가 상호작용할 수 있게 만드는 언어이다.
기존 정적인 HTML 페이지에서 사용자의 선택으로 페이지의 동적인 변화를 주는 것이 목적이다.

  1. document.write("~~")

    : 스크립트 태그 안에 들어가며, 동적인 텍스트 결과값을 나타냅니다.

  2. input type="~~" 이벤트="~~"

    : 버튼이나 텍스트 등의 타입을 정하면, 이벤트에 해당하는 행동을 했을 시 어떤 반응을 나타냅니다.

  3. 비교연산자 ===

    : 좌항과 우항의 값이 같을 경우 true, 다를 경우 false를 출력한다. (여기서 true와 false를 Boolean이라고 한다.)

  4. 비교연산자 < , >

    : 좌항과 우항의 값을 부등호로 계산해 맞을 경우 true, 틀릴 경우 false를 출력한다.

  5. 조건문 if(t/f) {~~} else {~~}

    : if문의 계산이 true인 경우 바로 뒤의 값을 출력하고, 틀릴 경우 else 뒤의 값을 출력한다.

  6. 변수 창출 var (target)

    : 태그의 중복을 줄이기 위해 하나의 변수를 만들어 냅니다.

  7. 중복 제거 if(this)

    : if문에 this를 사용함으로써 id 태그를 지울 수 있습니다.

  8. 반복문 while(t/f) {~~}

    : 조건이 참일 경우, 지정된 태그문을 반복합니다. ex) 변수var x = 0; while (x < 3) {태그문}

  9. document.querySelector('~~')

    : 괄호 안에 있는 태그가 달린 선택자 하나를 출력합니다.
