박스 모델

이 페이지는 (박스 모델)을 학습하기 위한 연습용 페이지입니다.

  1. border:~~

    : 테두리의 속성을 변경합니다. (~~px, solid, red 등등)

  2. padding:~~

    : 텍스트와 테두리 사이의 공간을 조절합니다.

  3. margin:~~

    : 블럭 간의 거리를 조절합니다.

  4. display:~~

    : 해당 태그문의 속성을 부여합니다. (block,inline 등등)

  5. width:~~

    : 태그의 크기를 변경합니다.

  6. block, inline

    : 블럭은 전체 크기의 태그를 의미하고, 인라인은 자기자신 크기의 태그를 의미합니다.
