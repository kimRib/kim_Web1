CSS란 무엇인가?


코딩 배우기 두 번째, 이 페이지는 CSS에 대한 페이지이며 연습용입니다.

  1. a {color:~~}

    : a태그가 붙은 모든 코드의 색상을 변경합니다.

  2. a {text-decoration: ~~}

    : a태그가 붙은 모든 코드의 텍스트 꾸밈을 변경합니다.

  3. a {text-align: ~~}

    : a태그가 붙은 모든 코드의 텍스트 정렬을 변경합니다.

  4. a {font-size: ~~}

    : a태그가 붙은 모든 코드의 텍스트 크기를 변경합니다.

  5. style="color:~~"

    : HTML 태그문 안에 사용하는 속성 코드로, 해당 텍스트의 색상을 변경합니다.

  6. style="text-decoration:~~"

    : HTML 태그문 안에 사용하는 속성 코드로, 해당 텍스트의 꾸밈을 변경합니다.

  7. class=""

    : HTML 태그문 안에 사용하는 속성 코드로, 태그들의 그룹을 지정해줍니다. css문으로 사용시 해당 그룹명 앞에 .을 붙입니다.

  8. id=""

    : HTML 태그문 안에 사용하는 속성 코드로, 태그들의 그룹을 지정해주고 최우선 순위를 부여합니다. css문으로 사용시 해당 그룹명 앞에 #을 붙입니다. 단 한번만 사용 가능.

  9. div

    : 디자인을 위해 쓰이는 태그로, 블럭 레벨 단위로 텍스트를 지정해줍니다.

  10. span

    : 디자인을 위해 쓰이는 태그로, 인라인 레벨 단위로 텍스트를 지정해줍니다.

  11. media 쿼리

    : style 문 아래에 @media 로 사용되는 태그로, 반응형 미디어 디자인을 다루기 위해 사용됩니다.
    ex) 화면 크기가 800px이하일 때 본문과 목록의 위치를 조정할 경우, max나 min을 사용해 쿼리문 제작.
